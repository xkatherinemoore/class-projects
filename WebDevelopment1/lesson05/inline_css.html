Font Families

The font family (or typeface) is the name of a specific design of characters. All fonts fall into one of the generic font families that are grouped based on similar characteristics (W3Schools, 2023). It's important to note that fonts are rendered on a web page using the downloaded fonts on a user's computer. For this reason, multiple fonts are typically specified in the font-family property with a generic family last to ensure a similar look.

Color Methods

There are three main color methods to apply color to a web page: RGB, hexadecimal, and the color name.

Color Method Explanation Format Example
RGB color Color is written in a format that utilizes the red, green, and blue color channels which are then blended to create a single color. Color channels can range from 0-255. color: rgb(red_value,green_value,blue_value); This text uses the CSS code "color: rgb(35, 221, 69);".
Hexadecimal color Color is written in hexadecimal format (0-9, A-F). Hexadecimal numbers represent the color channels red (R), green (G), and blue (B) in the format RRGGBB. In CSS, it must be preceded by a hash/pound sign (#). color: #RRGGBB; This text uses the CSS code "color: #883b46;".
Color names Color is written using a CSS accepted (Web-safe) color name. There are significantly fewer (only 140) accepted color names than there are RGB/Hexadecimal colors. A list of accepted color names can be found here. Note that if a color name is more than one word, spaces are removed. color: color_name; This text is slateblue.
